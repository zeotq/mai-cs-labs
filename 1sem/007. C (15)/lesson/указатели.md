# Указатели

```c
int a = 5;
int * p = &a;

int vals[3] = {1, 2, 3};
//  vals[i] <=> *(vals + i)
//  i * sizeof(int)
```

```c
void *p = &a;
//  p + i  - смещение на i байт
```

```c
int fact(int n);
```
n | fact(n)
--|--------
1 | 1
2 | 2
3 | 6
4 | 24
n | n!
---
<br>


```c
int (*p_f)(int) = fact;

p_f(n);
(*p_f)(n); //  эквивалетные выводы
```
